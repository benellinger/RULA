######################################


To run this application move to folder "exe.winamd/". Double click on "app.exe". A command prompt will appear.
The script will automatically load the data inside the folder "data/". Please make sure to only have ONE data-file at the time inside the folder. The File name NEEDs to be "data.csv" File format needs to be *.csv

This application runs on WINDOWS-64bit systems only.

This application is made for original German data files, exported from Noraxons MR3 software. The current compatible version of mr3 is mr3.10.64. For future versions an implementation of mr3.12 data format is planned.
